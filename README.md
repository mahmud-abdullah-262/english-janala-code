## 📖 English Janala  

**English Janala** is a simple and user-friendly web application designed to help people learn English words in an organized and effective way.  

The goal of this project was to create a smooth learning experience where users can explore vocabulary easily and understand each word with helpful details. Words are categorized into lessons, making the learning process structured and engaging.  

While building this project, I focused on improving **data fetching, UI rendering, and handling dynamic content using JavaScript**.

## 🛠️ Technologies Used  

- ⚡ Vanilla JavaScript (ES6)  
- 🌐 HTML5  
- 🎨 CSS3  
- 💨 Tailwind CSS  
- 🌼 DaisyUI  
- 🔗 API Integration (HTTP Requests)  
- 🧠 DOM Manipulation  
- 📡 Data Fetching  
- 🎯 Complex UI Handling  
- 📐 Figma (Design Reference)  

---

## ✨ Key Features  

- 📚 Words organized into **different lessons**  
- 🔊 Each word includes **pronunciation (audio support)**  
- 📖 Displays **meaning, synonyms, and example usage**  
- 🔍 **Search functionality** to quickly find words  
- 🎧 Listen to word pronunciation  
- 🎨 Clean and simple UI for better learning experience

## 📦 Dependencies
